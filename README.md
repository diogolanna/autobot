# AutoBOT

# Contributing

1. Fork this project and clone
2. Set remote upstream
```
git remote add upstream https://github.com/gamabot/autobot
```
3. Sync branches
```
git fetch upstream
```
4. Merge
```
git merge upstream/master
```

# How to run?

1. Clone this project
2. Go to project dir and install dependencies
```
npm install
```
3. run
```
node index.js
```
